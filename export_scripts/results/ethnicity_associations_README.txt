ethnicity: the ethnic group results were replicated in
phenotype: the name of the phenotype
n_individuals_tested: the number of individuals included in the association testing i.e. unrelated high-quality samples with the phenotype
measurement_visits: the different visits measurements were drawn (always choosing the first if measurements were taken at multiple visits). If more than one, then a categorical covariate is added for each beyond the first. categorical_covariates: categorical covariates included in the association tests other than those for visits (see above). Covariates that all aassume all values are listed by name, covariates that assume only some values are listed as pairs (covariate name, tuple of assumed values)
